Meowic 是一個音樂網頁應用。目前處於 **第一版開發階段 (v1.0)**，旨在透過最新的 Angular 技術棧實踐高效的前端架構。

## 🛠 技術特點
*   **核心框架**：**Angular 21+**
*   **表單架構**：**Reactive Forms**
*   **模板邏輯**：全面採用現代化 **`@if`** / **`@for`** 語法
*   **開發理念**：
    *   **語意化優先**：HTML 標籤與變數命名皆具備清晰意涵。
    *   **易讀與維護**：嚴格拆解元件職責，確保代碼結構簡潔。
